Mining Software

If you are a crypto mining pro then you'll probably consider one of the following options. For those who are new to crypto mining, we recommend studying this article before starting. The article talks about Monero (XMR) but approach is the same with XHV mining since it's using a version of CryptoNight algorithm and XHV is also based on CryptoNote.
Good selection of miner software with links and descriptions can be found on cryptunit.com's download page.

XMR-Stak is a universal Stratum pool miner. This miner supports CPUs, AMD and NVIDIA gpus and can be used to mine the CryptoNote currencies like Haven, Monero and Aeon.
Example configs: pools.txt, config.txt, cpu.txt (4 core i7 CPU), nvidia.txt (notebook GPU), amd.txt (5x RADEON RX 480)
XMRig-CPU - fastest CPU miner
download xmrig-2.6.4-msvc-win64.zip for Windows		 CPU GitHub
 Reddit
XMRig is a high performance Monero (XMR) CPU miner, with official support for Windows. Originally based on cpuminer-multi with heavy optimizations/rewrites and removing a lot of legacy code, since version 1.0.0 completely rewritten from scratch on C++.

Example config: config.json (4 core i7 CPU)
Parameters "cpu-affinity", "cpu-priority", "max-cpu-usage", "threads" are important for miner efficiency and should be selected accoridng to the processor architecture. Check here for more information.
